Multiplication Table Generator

Use the inputs below to create a multiplication table. You can use the numeric inputs (or the corresponding sliders below each) to define the range of the width and height of the table.

When you edit the dimensions of the table, you edit the table belonging to the currently open tab. You can press the "Tab" button to save this particular table to a new tab. You can delete individual or multiple tabs. To delete a single tab, press the 'x' button in the corner of the tab and it will be removed. To delete multiple tabs, select the checkboxes in the corners of the tabs you wish to remove, and then press "Delete Selected Tabs" and the tabs will be deleted.

This page was built using jQuery and Bootstrap CSS. jQuery handles all of the input validation and provides the framework for the tabs structure and the slider controls.
